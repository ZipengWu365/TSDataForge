# Support

For usage questions, open a GitHub issue and include:

- your Python version and operating system
- how you installed TSDataForge
- a minimal reproducible example
- the input file type or array shape you are working with
- the generated report or handoff bundle if it helps explain the issue

Before opening an issue, it is usually worth checking:

- [README.md](README.md)
- [docs/quickstart.md](docs/quickstart.md)
- [examples/README.md](examples/README.md)

For security issues, do not use the public issue tracker. Follow [SECURITY.md](SECURITY.md).
